【禁止事項清單】

  ✗ style="margin-top: 3rem"        → 改用 mb-12
  ✗ style="padding: 2rem"           → 改用 p-10 md:p-14
  ✗ style="background: #f5f5f5"     → 改用 bg-gray-50
  ✗ font-bold (標題)                → 改用 font-light
  ✗ 沒有響應式 class                → 要加 md: 和 lg:
  ✗ 混合舊章節樣式                  → 統一用 Tailwind
  ✗ 硬編碼色值 #...                 → 用 gray-* 色彩系統
  ✗ 自定義 CSS class                → 用 Tailwind utilities
